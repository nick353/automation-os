# Automation OS Vision

Automation OS is a same-Mac control panel for recurring Codex App work. The MVP focuses on visibility, lane isolation, approval grouping, and proof receipts so failed or partial automations are easier to resume.

The main AI lanes are Codex CLI, Codex App, and ChatGPT subscription-backed operation. OpenAI API usage is intentionally optional and documented as a future adapter, not a requirement for install, test, import, or demo runs.

The control panel should answer three questions quickly: what is running, what is blocked, and what proof exists.
