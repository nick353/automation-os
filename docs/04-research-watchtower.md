# Research Watchtower

Research Watchtower ingests daily knowledge into `advisor_events`. The Opportunistic Advisor is designed to propose help only when a run context matches a trigger.

Seed knowledge includes Runway MCP alternatives:

- Playwright CLI fallback lane.
- Chrome CDP upload/export observation.
- local export watcher for proof capture.

The point is not constant advice. The point is timely advice when the current blocker matches known tool drift or a better local route.
